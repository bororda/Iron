# Iron

Soldering Iron v3 on ATTiny

usefull page: https://www.instructables.com/id/Getting-Started-With-OLED-Displays/

How to programm:
1. Go to File/ Settings
2. Board manager link: http://drazzy.com/package_drazzy.com_index.json
3. Go to Instruments/ Boards/ Manage Boards, find and install ATTinyCore by Spence Konde
4. Go to Instruments/ Boards/ and select ATtiny25/45/85
5. Set Clock Freq to 8MHz
6. Programm with another arduino (e.g. Nano) by choosing Arduino as ISP in Programmator menu
